# HoTTServer
Arduino library to send telemetry data to Graupner HoTT-recievers

### Required hardware:
* Arduino Pro Mini 328, 16Mhz, 5V
* 1x resistor 1k5 to 4k7 Ohm (SMD 0805 fits perfect)
